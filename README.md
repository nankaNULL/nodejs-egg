# nodejs-egg
egg框架学习，根据开发文档，尝试搭建

===========================================

#### 项目结构

```text

├── config // 配置文件
│     └── config.default.js
│
├── app //项目的主要目录
│     ├── controller //控制器
│     ├── public //静态文件
│     │     ├── js 
│     │     └── css 
│     ├── view //页面
│     ├── extend //扩展插件
│     ├── service //http请求
│     └── router.js //路由
│
└── package.json //node相关环境的配置文件

```
